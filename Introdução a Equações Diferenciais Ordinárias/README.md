# Introdução às Equações Diferenciais Ordinárias (EDO)

Material de apoio para aulas introdutórias de Equações Diferenciais Ordinárias (EDO).

Este repositório reúne slides desenvolvidos em Markdown, com versões em PDF para visualização e uso em aula.

---

## 📚 Estrutura

As aulas estão organizadas pelas pastas pdf e markdown

---

## 🎯 Objetivo

Este material tem como objetivo:

* Introduzir os conceitos fundamentais de EDO
* Servir como apoio didático em sala de aula
* Disponibilizar conteúdo aberto e reutilizável

---

## 🛠️ Tecnologias utilizadas

* Markdown
* LaTeX (para notação matemática)
* GitHub (versionamento e distribuição)
